# Webcam Smoke Effect — Design Spec

## Overview

웹캠으로 담배 피는 손 제스처를 감지하여 화면에 연기 효과를 렌더링하는 웹 애플리케이션.
검정 배경 위에 흰 연기가 대비되는 시각적 임팩트를 핵심으로 한다.

**용도:** 개인 재미/장난 + 인터랙티브 아트/전시
**타겟 환경:** 데스크톱 Chrome / Edge (모바일은 스코프 외)

## Tech Stack

- **손 인식:** `@mediapipe/hands@0.4.1675469240` (CDN, 버전 고정)
- **얼굴 인식:** `@mediapipe/face_mesh@0.4.1633559619` (CDN, 버전 고정)
- **노이즈:** simplex-noise 라이브러리 CDN (`simplex-noise@4.0.1`) — 사실적 모드 연기 흔들림용
- **연기 렌더링:** Canvas 2D 파티클 시스템 (사전 렌더링 스프라이트 방식)
- **빌드 도구:** 없음 — 순수 HTML + JS, CDN 의존
- **실행 방법:** 로컬 서버 필요 (`python -m http.server` 또는 `npx serve`) — `file://` 프로토콜은 CORS/getUserMedia 이슈 발생 가능

## Architecture

단일 페이지 웹앱.

```
index.html          ← 메인 페이지 (검정 배경, 캔버스, 웹캠 PIP)
├── js/
│   ├── app.js      ← 앱 초기화, 메인 루프 (requestAnimationFrame)
│   ├── hand.js     ← MediaPipe Hands 초기화 & 제스처 판별
│   ├── face.js     ← MediaPipe Face Mesh & 입 위치 추출
│   ├── smoke.js    ← 파티클 시스템 (생성, 물리, 렌더링)
│   └── modes.js    ← 사실적/아트 모드 프리셋 관리
└── style.css       ← 최소 스타일링
```

### 웹캠 설정

- 요청 해상도: `640x480` (성능과 품질 균형)
- MediaPipe 랜드마크는 정규화 좌표(0~1) → 캔버스 좌표 매핑: `canvasX = canvasWidth * (1 - landmark.x)`, `canvasY = canvasHeight * landmark.y` (좌우 반전 포함)
- 웹캠 PIP는 CSS `transform: scaleX(-1)`로 미러링 (자연스러운 거울 느낌)

### Main Loop

1. 웹캠 프레임 캡처
2. MediaPipe Hands에 프레임 전달 (매 프레임)
3. MediaPipe Face Mesh에 프레임 전달 (3프레임마다 1회 — 성능 최적화)
4. 제스처 판별: 담배 포즈 감지
5. 손 위치 vs 입 위치 비교 → 연기 발생 지점 결정
6. 파티클 생성 & 업데이트 & 렌더링
7. 웹캠 PIP 렌더링

### 손 감지: 단일 손만 사용

- MediaPipe Hands `maxNumHands: 1` 설정
- 양손 감지는 스코프 외

## Gesture Detection

### 담배 포즈 감지 조건

MediaPipe Hands 랜드마크 기반. 팜 너비(palm width)는 손목(0번)과 중지 MCP(9번) 사이 거리로 정의.

- **검지 끝(8번)과 중지 끝(12번)** 사이 거리: `0.03 < dist / palmWidth < 0.12` (조절 가능한 상수)
- **검지-중지가 펴져 있음:** 각 손가락 끝이 해당 PIP 관절보다 위에 위치
- **약지, 새끼손가락 접혀 있음:** 각 손가락 끝이 해당 PIP 관절보다 아래에 위치
- **엄지:** 엄지 끝(4번)이 검지 MCP(5번)보다 바깥으로 벌어지지 않은 상태 (완전히 편 손 false positive 방지)
- 모든 조건 충족 시 → "담배 포즈" 활성화

### 입 근처 판별

Face Mesh 기준:
- 얼굴 높이(faceHeight) = 이마(10번)과 턱(152번) 사이 거리
- 손끝(검지-중지 중간점)과 입(13번, 윗입술 중앙) 사이 거리
- **입 근처 기준:** `dist < 0.35 * faceHeight`

### 안정성 처리

- 3프레임 연속 감지 시에만 활성화 (깜빡임 방지)
- 3프레임 연속 미감지 시에만 비활성화

### 연기 발생 상태 머신

| 상태 | 연기 | 설명 |
|------|------|------|
| 담배 포즈 + 손이 입에서 먼 경우 | 손끝에서 잔잔한 연기 | 담배 끝에서 피어오르는 느낌 |
| 담배 포즈 + 손이 입 근처 | 연기 없음 (흡입 중) | 빨아들이는 동작 |
| 담배 포즈 + 손이 입에서 떨어지는 순간 | 입에서 연기 내뿜기 | 입에서 많은 파티클이 퍼져나감 |
| 포즈 해제 | 파티클 생성 중단 | 기존 연기 서서히 소멸 |

**내뿜기 트리거 조건:**
1. 손이 입 근처에 0.3초 이상 체류
2. 손이 입에서 `> 0.15 * faceHeight` 이상 이동 (미세 떨림과 구분)
3. 두 조건 모두 충족 시 발동
4. 입 위치에서 파티클이 퍼지며 위로 상승

## Particle System

### 렌더링 방식: 사전 렌더링 스프라이트

**Canvas 2D `filter: blur()`는 사용하지 않는다** (프레임당 적용 시 60fps 불가능).
대신:
- 앱 초기화 시 오프스크린 캔버스에 radial-gradient 원형 스프라이트를 미리 렌더링
- 각 파티클은 이 스프라이트를 `drawImage()`로 찍는 방식 → GPU 친화적, 매우 빠름
- 모드별로 다른 스프라이트 세트 (흰색/컬러)

### 파티클 속성

- 위치(x, y), 속도(vx, vy), 크기, 투명도(alpha), 수명(lifetime)
- `globalCompositeOperation: 'screen'` + 오프스크린 캔버스 합성 (additive `lighter`는 순백 포화 문제)
- 초기 alpha: 0.02~0.08 (낮게 시작하여 겹침 시 자연스러운 볼류감)

### 모드별 차이

| | 사실적 모드 | 아트 모드 |
|---|-----------|---------|
| 색상 | 흰색~연회색 | 네온/그라데이션 (시안, 마젠타, 보라 등) |
| 움직임 | 자연스러운 상승 + 좌우 흔들림 (simplex noise) | 소용돌이, 나선형, 파동 패턴 |
| 크기 | 작게 시작 → 서서히 커지며 흐려짐 | 다양한 크기 + 맥동 효과 |
| 파티클 수 | ~100개 | ~200개 |
| 스프라이트 | 부드러운 흰색 radial gradient | 컬러 radial gradient + 글로우 |

### 모드 전환 동작

기존 파티클은 현재 모드 속성 유지하며 자연 소멸. 새로 생성되는 파티클만 새 모드 적용. 별도의 크로스페이드 없음.

### 발생 위치별 차이

- **손끝:** 파티클이 위로 천천히 상승, 적은 양, 잔잔함
- **입 (내뿜기):** 파티클이 입에서 전방으로 빠르게 퍼지며 위로 상승, 많은 양, 역동적

### 성능 최적화

- 파티클 풀링 (객체 재사용, GC 방지)
- 수명 다한 파티클 즉시 풀 반환
- 최대 파티클 수 제한: 300개
- Face Mesh는 3프레임마다 1회 추론 (입 위치는 프레임 간 보간)

## UI Layout

- **배경:** 전체 화면 검정 (`#000`)
- **중앙:** Canvas (연기 렌더링, 화면 전체 크기)
- **우측 상단:** 웹캠 PIP (작은 라운드 사각형, 반투명 테두리, 미러링)
- **좌측 하단:** 모드 전환 버튼 (Realistic / Artistic) — 작고 미니멀

### 키보드 단축키

- `Space` 또는 `M`: 모드 전환
- `F`: 풀스크린 토글
- `H`: 웹캠 PIP 숨기기/보이기

## Error Handling

- 웹캠 접근 거부 시: 화면에 안내 메시지 표시
- MediaPipe 로딩 실패 시: CDN 에러 메시지 + 재시도 안내
- 손/얼굴 미감지 시: 조용히 대기 (에러 아님)

## Testing

- 제스처 감지 정확도: 다양한 손 각도/거리에서 수동 테스트
- 파티클 성능: 60fps 유지 확인 (Chrome DevTools Performance)
- 모드 전환: 기존 파티클 유지 + 새 파티클 새 모드 적용 확인
- 브라우저 호환성: Chrome, Edge (MediaPipe 지원 브라우저)
